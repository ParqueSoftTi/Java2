En este codelab, desarrollaremos una aplicación de gestión de productos utilizando Spring Framework. La aplicación permitirá a los usuarios crear, leer, actualizar y eliminar productos, así como realizar consultas avanzadas utilizando JPQL. También implementaremos una API REST para acceder a los datos de productos y documentaremos la API utilizando Swagger.

Configuraremos una base de datos H2 para almacenar la información de productos. Utilizaremos el archivo application.properties para configurar la conexión a la base de datos.

spring.datasource.url=jdbc:h2:mem:testdb
spring.datasource.driverClassName=org.h2.Driver
spring.datasource.username=sa
spring.datasource.password=password
spring.jpa.database-platform=org.hibernate.dialect.H2Dialect
spring.h2.console.enabled=true

│   │       └── example/

│   │               ├── repository/
│   │               └── service/

    └── java/

                └── productmanagement/
